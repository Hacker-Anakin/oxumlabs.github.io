Virtual Execution Node Intermediate Language (VenIR)
Introducing Ven, an innovative compiler infrastructure designed to empower you to create your own compilers! Ven stands out by producing exceptionally fast and lightweight binaries for the three major platforms: MacOS, Linux, and Windows. What sets Ven apart is its intelligent conversion of IR code into tokens, followed by extremely aggressive code elimination techniques—not limited to just dead code. Ven's primary goal is to strip down code as much as possible, ensuring that only the essential elements remain. This meticulous process results in minimal and highly efficient binaries that perform at peak speed, enhancing your development experience.
You can read more about Ven on its GitHub repo here!.

Jock
Jock is a file encryption software NOT DESIGNED FOR PROFESSIONAL USE (YET) using which, you can easily encrypt your files with a password and recover them when needed. When done using it, you can lock them back up! Jock is available for Windows and Linux and can be built from source for macOS. Find the Jock repository here.
